使用FFmpeg将多张图片拼接成视频或制作幻灯片

FFmpeg图片处理功能

FFmpeg可以将多张图片拼接成视频、创建幻灯片或生成动态相册。以下是一些常用的图片处理命令：

基本图片转视频

# 将一系列图片转换为视频（按数字顺序） ffmpeg -framerate 24 -i image%03d.jpg output.mp4 # 指定图片格式和帧率 ffmpeg -framerate 30 -pattern_type glob -i "*.png" output.mp4

创建幻灯片效果

# 创建带有过渡效果的幻灯片 ffmpeg -framerate 1/3 -i img%03d.jpg -vf "fps=25,format=yuv420p" -c:v libx264 -crf 23 slideshow.mp4 # 添加缩放平移效果（Ken Burns效应） ffmpeg -i img%03d.jpg -vf "zoompan=z='min(zoom+0.0015,1.5)':d=125:x='if(gte(zoom,1.5),x,x+1)':y='y+1':s=1280x720" -c:v libx264 output.mp4

图片拼接与网格布局

# 将4张图片拼接成2x2网格 ffmpeg -i input1.jpg -i input2.jpg -i input3.jpg -i input4.jpg \ -filter_complex "[0:v][1:v]hstack=inputs=2[top];[2:v][3:v]hstack=inputs=2[bottom];[top][bottom]vstack=inputs=2[v]" \ -map "[v]" output.jpg # 创建3x3图片网格 ffmpeg -i input1.jpg -i input2.jpg -i input3.jpg -i input4.jpg -i input5.jpg \ -i input6.jpg -i input7.jpg -i input8.jpg -i input9.jpg \ -filter_complex "[0:v][1:v][2:v]hstack=inputs=3[top];[3:v][4:v][5:v]hstack=inputs=3[middle];[6:v][7:v][8:v]hstack=inputs=3[bottom];[top][middle][bottom]vstack=inputs=3[v]" \ -map "[v]" output.jpg

使用说明

提示

1. 确保图片命名有规律（如img001.jpg, img002.jpg）或使用通配符

2. 对于幻灯片，较低的帧率可以让每张图片显示更长时间

3. 使用CRF值控制视频质量：18-28是常用范围，23是默认值

高级示例

创建带有淡入淡出效果的幻灯片：

ffmpeg -framerate 1/5 -i img%03d.jpg -vf "fps=25,format=yuv420p, \ scale=1280:720:force_original_aspect_ratio=decrease,pad=1280:720:(ow-iw)/2:(oh-ih)/2, \ fade=t=in:st=0:d=1,fade=t=out:st=4:d=1" \ -c:v libx264 -crf 23 -preset medium slideshow_with_fade.mp4

注意事项

1. 确保所有图片尺寸相同或使用scale过滤器统一尺寸

2. 图片数量较多时，拼接可能需要较长时间

3. 某些编码器可能需要额外安装

常用参数

主要参数

  • -framerate 输入帧率
  • -pattern_type 模式类型
  • -i 输入文件
  • -c:v 视频编码器
  • -crf 恒定质量因子

输出格式

  • .mp4 H.264/H.265
  • .webm VP8/VP9
  • .mov ProRes
  • .gif 动画GIF

网格布局参数

hstack - 水平拼接图片

vstack - 垂直拼接图片

xstack - 复杂网格布局

高级过滤器

zoompan - 创建Ken Burns效果

fade - 添加淡入淡出效果

overlay - 图片叠加

scale - 调整图片尺寸
